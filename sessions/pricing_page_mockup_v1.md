# 구독 플랜 비교 페이지 (Pricing Plan Comparison Page) Mockup v1

## 📐 디자인 원칙 및 목표
*   **목표:** 사용자가 자신의 '현재 리스크 레벨'을 인지하게 하고, 그에 맞는 최적의 솔루션(상위 티어)으로 이동하도록 유도한다.
*   **톤앤매너:** 권위적(Authoritative), 전문적(Technical), 명료함(Clarity). 감성적 어필은 배제하고 오직 '재무적 리스크'와 '투자 대비 회피 손실액 (Avoided Loss ROI)'에 초점을 맞춘다.
*   **컬러 팔레트:**
    *   Primary Dark Blue: `#0A1931` (헤더, 배경 강조)
    *   Secondary Gray: `#F4F6F8` (콘텐츠 섹션 배경)
    *   Text/Body: `#2C3E50` (짙은 네이비 그레이)
    *   Accent Color (CTA/Highlight): `#B388FF` (신뢰도를 주는 밝은 보라색 계열, 핵심 가치 강조)

## 🖥️ 섹션별 와이어프레임 및 구성 요소

### 1. 히어로 섹션 & 문제 재정의 (The Pain Point Reiteration)
*   **배경:** 다크 블루 (`#0A1931`) 그라디언트 또는 패턴 배경.
*   **헤드라인 (H1):** 당신의 비즈니스는 어떤 리스크를 회피하고 있습니까? (Avoided Loss: 측정되지 않은 손실을 방치하지 마십시오.)
    *   *(설명: '위협 지적'으로 시작하여 사용자가 본질적인 문제에 집중하게 만듭니다.)*
*   **서브 카피:** 단순한 규정 준수를 넘어, 미래의 불확실성을 예측하고 재무적으로 방어하는 아키텍처가 필요합니다. 저희 시스템은 '위험 발생 가능성'을 구체적인 손실 금액으로 계산하여 최적의 보호막을 설계합니다.
*   **CTA (메인):** 🛡️ 리스크 진단 보고서 받기 (Lead Magnet 유도)

### 3. 가치 증명 섹션: 왜 Pro인가? (The Upsell Nudge - Logic Flow)
*   **목표:** Basic과 Pro의 결정적인 차이점(Pain Point 해결 능력)을 극대화하여 Pro를 선택하도록 유도합니다.
*   **제목 (H2):** '준수'만으로는 충분하지 않습니다: 예측적 위험 관리로 나아가야 할 이유.
*   **콘텐츠:** 3단계의 논리 흐름으로 구성된 비교 인포그래픽.

| 기능/가치 | Basic (Compliance) | Pro (Assurance) | 차이점과 가치 (The Value Jump) |
| :--- | :--- | :--- | :--- |
| **위험 감지** | *Pass/Fail* 체크리스트 기반의 사후 보고. | $\text{A} \rightarrow \text{B} \rightarrow \text{C}$ 시나리오 기반의 예측 분석. | 🚨 **(차이점):** 단순 기록에서 '미래를 읽는 능력'으로 진화합니다. |
| **손실액 측정** | 일반적인 규제 벌금 예상치 제시. | 데이터 모델링을 통한 **최적 회피 손실액 ($)** 정량 제시. | 💰 **(차이점):** 모호한 '위험' 개념이 구체적인 '재무 리스크'로 치환됩니다. |
| **시스템 연동** | 웹 기반의 단일 플랫폼 사용. | API 게이트웨이를 통한 사내 시스템 완전 통합. | 🌐 **(차이점):** 사일로화된 데이터에서 전사적 비즈니스 운영 체계로 확장합니다. |

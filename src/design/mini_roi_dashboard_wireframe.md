# Mini ROI Simulator - Critical State Dashboard Wireframe v1.0
## 🎨 Design Goal & Tone:
*   **Goal:** 위기감 극대화와 시스템적 신뢰성 동시 확보. (공포 → 완벽한 통제)
*   **Tone:** Systemic, Authoritative, Report-like. (기술 문서/사고 보고서 스타일)

## 🎨 Color Palette & Typography:
*   Primary BG: `#1A237E` (Deep Navy Blue)
*   Secondary Warning: `#FFB300` (Amber Gold)
*   Critical Error: `#D32F2F` (Signal Red)
*   Text/Data: `#E0E0E0` (Light Gray)
*   Font: Pretendard / Noto Sans KR (Clean, Technical San-serif)

## 🖥️ Layout Structure (16:9 Mockup):

### 🔵 [HEADER - System Control Panel]
*   **Element:** Logo | Mini ROI Simulator v1.2.0 | **[STATUS: CRITICAL VIOLATION DETECTED]** (`#D32F2F` Bold)
*   **Design Note:** 배경에 미세한 회로 기판(Circuit Board) 패턴의 그라데이션을 오버레이하여 기술적 깊이를 부여합니다.

### 🚨 [SECTION A: Immediate Threat Summary - The Impact]
*   **Layout:** 3-Column Card Grid (Highly Visible, Red Focus)
*   **Card 1: Violation Type**
    *   Icon: ⚠️ (빨간 경고 삼각형)
    *   Title: Data Integrity Failure Detected
    *   Detail: 외부 규제 변화 반영 실패
*   **Card 2: Estimated Loss**
    *   Icon: 💸 (돈가방 아이콘)
    *   Title: Potential Financial Loss Estimate
    *   Value: **₩XX억 Y천만 원** (`#D32F2F`, Bold, Largest Font Size)
*   **Card 3: Action Deadline**
    *   Icon: ⏳ (시계 아이콘)
    *   Title: Required Mitigation Window
    *   Detail: D-Day: 48 Hours

### 📑 [SECTION B: Detailed Error Report - The Authority Core]
*   **Layout:** Formal Technical Log / Table Structure.
*   **Header:** **SYSTEM ERROR REPORT // Timestamp: YYYY-MM-DD HH:MM:SS**
*   **Table Columns:** `Field Name` | `Value` | `Interpretation` | `Severity`
*   **Data Rows (Mock Data):**
    *   Error Code: `MINIROI_E403B` | N/A | 시스템 프로토콜 위반을 의미합니다. | 🔴 CRITICAL
    *   Violation Trigger: AI Legal Risk Modeling | Source API Overload & Regulatory Gap Detected | 특정 모듈의 외부 데이터 처리 한계를 초과했습니다. | 🔥 HIGH
    *   Root Cause: External Data Feed Latency Exceeded | 규제 업데이트 주기 미반영으로 인한 위험 계산 오류 발생. | 🚨 ALERT (Must be fixed)

### ✅ [SECTION C: Mitigation Strategy & CTA Funnel]
*   **Layout:** Step-by-step process flow diagram (1 → 2 → 3).
*   **Step 1: Diagnose (Analyze)**
    *   Action: 위험 요소 상세 분석 리포트 다운로드 (Secondary Button)
*   **Step 2: Solution Path (Consult)**
    *   Action: 전문가 컨설팅 요청하기 (Tertiary Link/Button)
*   **Step 3: System Restoration (The Goal)**
    *   Element: **[Premium Plan Upgrade Required]** (Large, Contrasting Button - e.g., Solid Deep Blue Background)

# ✍️ Naver Column Writer (네이버 블로그 칼럼 자율 작성기)

유튜브 트렌드 및 시뮬레이션 데이터를 유기적으로 가공하여, 전문 테크 비즈니스 에반젤리스트 어조의 네이버 블로그용 칼럼을 자율 작성합니다.

## 🛠️ 주요 기능
1. **트렌드 동적 주입**: `trend_sniper_report.md` 파일에서 최근 유튜브 알고리즘 트렌드를 실시간 파싱하여 본문에 반영합니다.
2. **비즈니스 에반젤리스트 페르소나**: 고도의 IT 비즈니스 통찰력을 담아 신뢰성 있는 전문 문체로 집필합니다.
3. **가드레일 성공 가치 전달**: Pydantic v2 가드레일 및 E2E 테스트 성공 스토리를 융합하여 정량적 리스크(Expected Loss) 감소 가치를 독자에게 입증합니다.
4. **자율 백업**: `naver_posts/` 폴더 하위에 날짜별(`post_YYYYMMDD_HHMM.md`) 마크다운 형태로 자율 백업합니다.
5. **오프라인 회복 탄력성(Mock Fallback)**: 로컬 LLM 서버 장애 시 명품 A급 사전에 기획된 모의 포스팅 본문으로 매끄럽게 자동 폴백하며 무결하게 `exit 0`으로 완료됩니다.

## 🚀 실행 방법
```bash
python _company/_agents/youtube/tools/naver_writer.py
```

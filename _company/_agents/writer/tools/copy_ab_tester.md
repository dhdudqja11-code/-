# ✍️ 마케팅 후크 자가 검증 및 가상 CTR 예측기 (Copy AB Tester)

카피라이터(Writer) 에이전트가 광고 문구, 블로그 포스팅 후크, 뉴스레터 제목 등을 작성한 뒤, 사장님께 승인 요청을 보내기 전 **텍스트 소구 패턴(NLP)** 및 **가상 타겟 반응도 시뮬레이션**을 통해 카피 매력과 심의 규정 준수 여부를 선제 감사(Audit)하는 마케팅 품질 검증 도구입니다.

---

## 💡 어떻게 도와주나요?

- **페르소나별 가상 클릭률(CTR) 연산**:
  - `2030 트렌디 직장인`, `4050 알뜰 부모`, `1인 창업가/소상공인` 성향에 따른 맞춤 클릭 매력도 시뮬레이션.
- **NLP 문장 소구 패턴 가산점 적용**:
  - 문장에 숫자 포함 여부 (+0.8%), 궁금증 질문 후크 (+0.5%), 명확한 행동 촉구(CTA) 키워드 (+0.6%), 타겟 매칭 단어 매칭 (+최대 1.2%) 등 정적 텍스트 패턴 정밀 가산.
  - 가독성 저하를 유발하는 극도의 장문(80자 초과) 감점 제어 (-0.6%).
- **과대 광고 사전 자가 필터링 (Risk Shield)**:
  - 심의 금지 어휘("100% 보장", "무조건 성공" 등)를 기계적으로 필터링하여 감점 및 자동 반려.

---

## ⚙️ 입력 매개변수 및 사용법

```powershell
python _company/_agents/writer/tools/copy_ab_tester.py --copy "<카피 내용>" [--persona <페르소나ID>]
```

### 대표 예시
* **1인 창업가 대상 카피 AB 테스트 실행**:
  ```powershell
  python _company/_agents/writer/tools/copy_ab_tester.py --copy "하루 2분만 투자해서 자동화로 월 500만원 벌어가는 성공 비결 무료 공개!" --persona SOLOPRENEUR
  ```
* **도구 자가 진단 E2E 예제 테스트**:
  ```powershell
  python _company/_agents/writer/tools/copy_ab_tester.py --test
  ```

## 🛠️ 카피라이터(Writer) 자율 행동 지침 (Agent Guideline)

1. **사전 의무 필터**: 사장님께 결재 텔레그램 카드를 올리기 전, 작성된 모든 주요 마케팅 후크 문구는 반드시 이 `copy_ab_tester.py`를 거쳐 사전에 자가 검증을 수행해야 한다.
2. **미달 시 자가 치유 (Self-Healing)**: 가상 CTR 결과가 합격점(기본 3.5%)에 미달하거나 심의 위반(`HIGH_RISK`)으로 반려(Exit 2)가 난 경우, 절대로 사장님께 결재를 올리지 말고 trace 오류 JSON 피드백을 해석하여 **문장 구조를 더 직관적이고 혜택 중심으로 다듬어 재검증을 즉각 구동**해야 한다.

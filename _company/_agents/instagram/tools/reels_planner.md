# 📷 Reels Planner (릴스/쇼츠 숏폼 대본 기획기)

최신 트렌드 분석 리포트 또는 마케팅 콘텐츠를 기반으로 인스타그램 릴스(Reels) 및 유튜브 쇼츠(Shorts)용 15~30초 초압축 고화질 숏폼 대본을 자율 기획합니다.

## 🛠️ 주요 기능
1. **트렌드 동적 연계**: `trend_sniper_report.md` 파일에서 트렌드 키워드 및 알고리즘 해킹 포인트를 추출하여 대본에 자동 반영합니다.
2. **초고속 숏폼 후킹**: 3초 이내에 시청 이탈을 방지할 수 있는 시각적/청각적 강력한 훅(Hook)을 설계합니다.
3. **타임라인별 매칭**: 시간(초) 단위로 비주얼 연출 지시, 나레이션 오디오 대본, 자막 카션을 입체적으로 매핑합니다.
4. **자율 백업**: `reels_scripts/` 폴더 하위에 날짜별(`script_YYYYMMDD_HHMM.md`) 마크다운 형태로 자율 백업합니다.
5. **오프라인 Mock Fallback**: LLM API 호출 불가 시 기획된 명품 모의 릴스 대본 및 트렌드 해시태그 패키지로 자동 폴백하여 `exit 0`으로 완료됩니다.

## 🚀 실행 방법
```bash
python _company/_agents/instagram/tools/reels_planner.py
```

2.  **Analyze Agent Outputs & Extract Key Data Points:**

    *   **Writer Output:**
        *   Goal: 15-20 minute YouTube script draft (3 topics).
        *   Content Detail: Script structure defined (Hook $\rightarrow$ Body $\rightarrow$ CTA).
        *   Key Data/Concepts: Three core risks identified (GDPR/Data Sovereignty, System Errors, AI IP Violation). The script emphasizes *quantifying 'Expected Loss Amount'* and citing legal basis.

    *   **Researcher Output:**
        *   Goal: Enhance script authority with hard data/precedents.
        *   Method: Researching specific, shocking global precedents for the three risks.
        *   Key Data/Concepts (The most critical part): Provides structured findings suitable for direct inclusion in the script. Examples include citing legal articles or high potential fine amounts ("최대 2천만 유로", "개인정보의 범위 확대"). *This confirms the data is ready.*

    *   **Designer Output:**

        *   Key Action 1 (Prototype): Updated '경고 메시지 컴포넌트' error logic using E2E stress test results and JSON structure. This means the *visual failure state* is ready.

# 🚀 문장 처방전 전용 텔레그램 자동화 작업 기획서

> **개요:** 기존 VS Code 확장 프로그램(ConnectAI) 개조 작업은 잠시 보류하고, 접근성이 가장 뛰어난 **텔레그램(Telegram) 봇**을 활용하여 1인 기업(문장 처방전)의 주문, 결제, 발행을 스마트폰으로 자동화하기 위한 작업 지시서입니다.

---

## 1. 텔레그램 봇의 핵심 역할 (비즈니스 시나리오)

### 💰 A. 실시간 결제 알림 (PayPal Webhook 연동)
- 페이팔에서 고객의 결제가 완료되는 즉시, 텔레그램 봇이 내 핸드폰으로 알림을 보냅니다.
- **알림 예시:** `"🔔 [결제 완료] 홍길동님이 '문장 처방전(깊은 beta)' 9,000원을 결제하셨습니다."`

### ✍️ B. 문장 처방전 원격 발행 지시
- 외부에서 핸드폰만 켜서 텔레그램 봇에게 명령어를 입력하면, 집에 있는 로컬 AI(LM Studio)가 글을 작성합니다.
- **명령 예시:** `/처방전 홍길동 최근 번아웃으로 힘들어함`
- **결과:** AI가 작성한 처방전 초안이 텔레그램 채팅방으로 텍스트 파일(.md) 형태로 전송됩니다.

### 🤖 C. 1인 기업 자동화 비서 (CS 및 알림)
- 고객의 문의 내역이나 오류 사항을 텔레그램으로 바로 받아보고, 버튼(승인/반려)을 눌러 즉각 대응합니다.

---

## 2. 향후 개발 로드맵 (To-Do List)

- [ ] **1단계: 텔레그램 봇 생성**
  - 텔레그램에서 `BotFather`를 검색하여 나만의 1인 기업 봇을 만들고 API 토큰(열쇠) 발급받기.
- [ ] **2단계: 백엔드 서버 연동 (`api_gateway.py`)**
  - 현재 작성 중인 페이팔 웹훅 코드에 텔레그램 전송 로직(API) 추가하기.
- [ ] **3단계: AI 글쓰기 연동 (LM Studio)**
  - 텔레그램에서 받은 키워드를 로컬 AI 서버로 넘겨서 특유의 심리학적 문체로 글을 작성하게 만드는 파이프라인 구축.
- [ ] **4단계: 최종 E2E 테스트**
  - 결제 발생 ➡️ 텔레그램 알림 ➡️ 원격 발행 지시 ➡️ 처방전 텔레그램 수신의 전체 사이클 검증.
